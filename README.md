# Oh My CodeAgent

一个基于 [tRPC-Agent-Go](https://github.com/trpc-group/trpc-agent-go) 框架开发的 Code Agent 项目。

## 项目简介

这是一个从 0 到 1 开始实现的 Code Agent，旨在探索和学习如何使用 tRPC-Agent-Go 框架构建智能代码助手。

项目目前处于起步阶段，将逐步实现一个能够帮助开发者进行代码相关任务的 AI Agent。

## 技术栈

- **语言**: Go
- **框架**: tRPC-Agent-Go

## 快速开始

```bash
# 克隆项目
git clone https://github.com/你的用户名/oh-my-codeagent.git
cd oh-my-codeagent

# 安装依赖
go mod download

# 运行项目
go run main.go
```
